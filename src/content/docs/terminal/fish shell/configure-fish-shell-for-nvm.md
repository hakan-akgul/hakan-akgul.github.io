1- Install oh my fish [omf](https://github.com/oh-my-fish/oh-my-fish)

```bash
curl https://raw.githubusercontent.com/oh-my-fish/oh-my-fish/master/bin/install | fish
```

2- Install nvm plugin via oh my fish

```bash
omf install nvm
```

3- That's all..

```bash
nvm --list
```
